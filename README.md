# MyCoins Analyzer — Automated Bukpot Extraction 🚀

Aplikasi automasi untuk ekstraksi data PDF Bukti Potong (Bukpot) PPh berbasis web. Dirancang untuk mempercepat input data keuangan secara presisi.

## 🌟 Overview
Proyek ini dibuat untuk menyelesaikan masalah input data manual dari dokumen perpajakan. Dengan menggunakan Python sebagai engine ekstraksi teks, data dikonversi dari PDF langsung ke Google Sheets secara otomatis.

## 🛠️ Arsitektur Sistem
- **Frontend**: HTML, Tailwind CSS, & JavaScript (Hosted on **Vercel**)
- **Backend**: FastAPI & PyPDF (Hosted on **Hugging Face Spaces**)
- **Integrasi**: Google Sheets API

## 📋 Fitur Utama
- **Login System**: Akses terbatas untuk keamanan data.
- **Auto-Extraction**: Mendeteksi No Bukpot, NPWP, Masa Pajak, & Nominal secara otomatis.
- **Direct Export**: Data langsung terkirim ke Google Sheets setelah di-upload.
- **Glassmorphism UI**: Tampilan modern dan responsif.

## 📖 Cara Penggunaan
1. Akses web melalui link [Link Vercel Kamu].
2. Login menggunakan akun admin.
3. Pilih file PDF Bukti Potong, lalu klik "Analisis".
4. Cek hasil ekstraksi pada tabel di bawah atau langsung di file Google Sheets.
